# Vocabulary Learning Tool with Ai asistant

If you learn external language - this tool for you!) 

This tool helps you learn and reinforce your personal vocabulary extracted from Google Docs using a Ai assistant

You have to a personal vocabulary like this

| English | Russian     |
| ------- | ----------- |
| absurd  | абсурд      |
| admire  | восхищаться |
| adore   | Обожать     |


The assistant will randomly select words and create sentences with them.

Your task is to provide the translation.

That's it! 

I made this app a habit, going through 5 sentences every day.
